# Product Images

Place all product images here.

Naming convention: {product-slug}-{variant}.png
Examples:
- /products/wpc-coffee-latte.png
- /products/bcaa-raspberry.png
- /products/creatine-unflavored.png

Usage in code: imageUrl = "/products/wpc-coffee-latte.png"
In admin panel: enter the path when creating a product variant.
